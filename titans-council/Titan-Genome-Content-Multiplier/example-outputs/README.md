# Example Outputs

This folder contains sample outputs showing what the Titan Genome produces. These are abbreviated examples to illustrate the structure and quality level -- not full deliverables.

## How to Use These Examples

1. **Review the structure** -- See how each project type is organized
2. **Check the quality bar** -- This is the standard the Critic and Approver agents enforce
3. **Use as reference** -- When reviewing your own outputs, compare against these examples

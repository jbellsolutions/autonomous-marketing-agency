## Quick Start

### 1. Configure Your Client

Open `CLIENT_CONFIG.md` and fill in your client details -- business name, niche, target audience, offer, voice/tone preferences. This file feeds every template in the system.

### 2. Install Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

### 3. Place Your Transcript

Drop your video transcript into the `source/` directory:

```bash
mkdir -p source
cp /path/to/your-transcript.md source/
```

### 4. Run Project 1 -- Content Map

Start with the Content Map. It analyzes your transcript and produces the master plan that all other projects build from.

```bash
claude
# Then: "Run Project 1 (Content Map) using the transcript in source/"
```

### 5. Pick Your Next Project

Consult the Buyer's Guide in `docs/` to choose which projects to run next based on your goals. Each project is independent -- run them in any order after the Content Map is complete.

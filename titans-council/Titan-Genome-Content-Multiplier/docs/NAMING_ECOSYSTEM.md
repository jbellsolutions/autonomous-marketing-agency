# Naming Ecosystem -- The Titan Genome

| Layer | Name | When To Use |
|---|---|---|
| Product Name | The Content Multiplier | External-facing product name. What it DOES. |
| Mechanism Name | The Titan Genome | Revealed in sales letters/VSLs. Why it WORKS. |
| Campaign Name | The One-Take Empire | Launch campaigns, ads, email subject lines |
| Community Name | Titans | Member identity, events (Titan Summit), certification (Titan Certified) |
| Internal Process | The Copy Genome | Technical reference to the decoded copywriting DNA system |
| Experience Metaphor | The Mastermind Table | Napoleon Hill's Invisible Counselors -- 9 legends at your table |

## Usage Guidelines

- **Homepage/landing page headline:** Lead with Content Multiplier, reveal Titan Genome as mechanism
- **Sales letter:** Open with Napoleon Hill / Mastermind Table, reveal Titan Genome mid-letter
- **Ads:** Use "One-Take Empire" hooks
- **Community:** "Titans" / "I have a table of nine"
- **Technical docs:** "The Copy Genome" for the AI pipeline
- **Events:** Titan Summit, Titan Live, Titan Certified

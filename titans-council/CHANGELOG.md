# Changelog — Titans of Direct Response Mastermind Council

All notable changes to this project are documented in this file.

---

## [1.0.0.0] — 2026-03-27

The current stable release. 18-agent council, 3-round deliberation, 3-agent Quality Gate, Content Multiplier with 13 project templates, and 2 completed client projects.

### The Full System

- **18 copywriter agents** running independently on every brief (15 direct response legends + 3 modern strategists)
- **59 sub-agents** across all 18 council members (3-5 per agent)
- **3-round council deliberation** (V1 synthesis, V2 cross-critique, V3 final consensus)
- **3-agent Quality Gate** (Creator scores /100, Critic scores /100, Approver makes ship/revise/send-back decision)
- **3 leadership agents** (Powell, Marquet, Sinek) available as invoke-only
- **Content Multiplier** with 13 project categories and 32 file templates
- **Titan Genome knowledge base** with 19,308 indexed entries across 650,000+ words
- **Client-facing README** and **ONE-PAGER.md** executive summary per project
- **Private GitHub repo publishing** per project
- **52 operational rules** governing agent behavior, quality, security, and communication

### Completed Client Projects

- Callum Crees — On Auto / AIOS (Revenue Leak Protocol)
- Darwin & Akilah — Let Soul Work (Inner Architecture Method)
- JBell Solutions — The Council Sells Itself (meta GTM project)

## [0.7.0] — Quality Gate + Pipeline Unification

- Added 3-agent Quality Gate: Creator QG, Critic QG, Approver QG
- Creator and Critic each score 10 dimensions out of 100
- Approver makes APPROVED / APPROVED WITH REVISIONS / SEND BACK decisions
- Unified the full pipeline from brief intake through QG approval
- Established the canonical final output path (round-3-final-approved.md)

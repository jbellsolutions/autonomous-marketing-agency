## The Mastermind Table — Where This Idea Comes From

In Chapter 10 of *Think and Grow Rich* — the chapter Napoleon Hill was most reluctant to publish — he made a confession that stunned his readers.

Every night before sleep, Hill would close his eyes and convene an imaginary meeting with the greatest minds in history. He called them his **"Invisible Counselors."** He would sit at the head of a long table, present his problems, ask for their counsel, and listen. Hill said this practice produced more breakthroughs and actionable ideas than anything else he ever did.

Most people read that chapter and think it's a visualization exercise.

**We read it and thought: what if you could actually build this?**

Not as imagination. Not as metaphor. *As technology.*

When you sit down with the Titans of Direct Response Mastermind Council, you are not using generic AI. You are pulling up a chair at the Mastermind Table. Eighteen agents — each one the decoded persuasion architecture of a legendary mind — deliberating around *your* offer, *your* message, *your* brand.

They don't just write copy. They argue with each other. They build on each other's ideas. They challenge weak thinking. They refine what's working. Three rounds of deliberation. Then a three-agent Quality Gate signs off on the final version before it ships.

2. **Open your terminal** — On a Mac, press `Cmd + Space`, type "Terminal", hit Enter. On Windows, search "PowerShell."

## FAQ

**Do I need to know how to code?**
No. You open Claude Code, talk to it, and it does everything. You just provide the brief about the offer.

**How long does a project take?**
About 30-45 minutes for the full pipeline (18 agents + 3 council rounds + Quality Gate + docs + GitHub publish).

**What's the Quality Gate?**
Three independent AI reviewers that check the council's work before it ships. The Creator validates creative quality, the Critic stress-tests for weaknesses, and the Approver makes the final ship/revise decision. Think of it as built-in QA.

**Can I run multiple projects for the same client?**
Yes. Each client gets their own folder, and each project within that client is separate. The agents start fresh every time — no bleed between projects.

**Can I share the output with a client?**
Yes. Each project gets its own private GitHub repo with a client-facing README. Share the repo URL. They don't need to know how to code — the README walks them through everything with clickable links.

**What about the leadership agents?**
They don't run automatically. Say "add Simon Sinek" or "run the leadership agents" and they'll join the analysis. They're separate from the copywriter council and Quality Gate.

**Can I add my own agents?**
Yes. Create a new `.md` file in `agents/` following the same format as the existing ones. The system picks up all agents from that folder automatically.

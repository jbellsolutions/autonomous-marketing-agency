## What Gets Produced

When this project runs through the Titans pipeline, here's what comes out:

| # | Output | Description |
|---|--------|-------------|
| 1 | **18 Individual Copywriter Takes** | Each legend independently analyses the offer and produces positioning, mechanism, headlines, hooks, sample copy, and strategy |
| 2 | **Council Round 1 — V1** | All 18 takes synthesized into a first integrated version |
| 3 | **Council Round 2 — V2** | V1 critiqued by every copywriter and refined |
| 4 | **Council Round 3 — Final** | Production-ready deliverable with positioning, brand voice, headlines, hooks, messaging, offer stack, mechanism build-out, go-to-market playbook, and sample copy portfolio |
| 5 | **ONE-PAGER.md** | Executive summary of the entire run |
| 6 | **README.md** | Client-facing front page with navigation and quick reference |
| 7 | **Private GitHub Repo** | Everything published and shareable |
